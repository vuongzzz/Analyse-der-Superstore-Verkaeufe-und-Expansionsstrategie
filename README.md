# Analyse-der-Superstore-Verkaeufe-und-Expansionsstrategie
Dieses Projekt nutzt Power BI, um weltweite Verkaufsdaten zu analysieren und die profitabelsten Märkte sowie Kundensegmente zu identifizieren. Basierend auf diesen Erkenntnissen werden datengesteuerte Strategien für die zukünftige Marktexpansion entwickelt.
